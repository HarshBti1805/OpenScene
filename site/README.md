# OpenScene Landing Page

Static marketing site for [OpenScene](https://github.com/HarshBti1805/OpenScene), built with Next.js (static export). It reuses the app's "Backlot" design language: the same tokens, typefaces (Big Shoulders Display, Inter, Courier Prime), and all three hand-tuned themes (Light, Dark, Midnight) plus system-follow.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs a fully static site to `out/` — deployable to GitHub Pages, Netlify, Cloudflare Pages, or any static host. Download buttons point at GitHub Releases.
